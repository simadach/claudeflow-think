  9800X3D（Zen 5）は B650E-I 出荷時の BIOS では動作しない場合がある。通常の BIOS 更新は「動作する CPU が手元にある」前提だが、9800X3D しかない場合は **BIOS FlashBack**（USB メモリから CPU なしで更新する機能）が必要になる。B650E-I が FlashBack に対応しているか、事前に ASUS 公式で確認し、更新用 USB を用意してから CPU を取り付けるべき。
  🔴 リスク: 高（確認せずに組み立てると POST すら通らない）
  🔧 確認コスト: 小（ASUS 公式ページを一読）
  > 返答: 組み立てにとくひつした難易度の過程をついかしたくはない。（返答確認済み）
  → 補記: 複雑な手順を追加したくないという方針を確認。FlashBack 自体は「USB を挿してパワーボタン長押し」のみで完結しモニター・GPU 不要だが、**USB の事前準備という工程が発生する**ことは変わらない。一切の事前準備を省きたいなら ①**7800X3D に戻す**（B650E-I 出荷 BIOS 対応済み）か ②**購入店で BIOS 更新を依頼する**（パソコン工房・ドスパラで対応可）という選択肢がある。idea.md でどちらの方針を取るか決めること。

- [ ] #003 **RTX 5070 の購入モデル（ASUS PRIME vs MSI VENTUS 2X）がまだ確定していないかもしれません** ⬅ モデル未決定
  A4-H2O に収まると確認されているのは ASUS PRIME（304mm）と MSI VENTUS 2X（236mm）。236mm は余裕が大きいが、小型クーラーは冷却性能・騒音が劣る傾向がある。SFF の密閉空間では GPU 温度がボトルネックになりやすいため、冷却重視なら ASUS PRIME（304mm）が無難。最終モデルを決めてから購入した方が後悔が少ない。
  🟡 リスク: 低（どちらも収まるが体験差あり）
  > 返答: （idea.md 購入前確認事項に選定ガイドラインを追記済み）（返答確認済み）

- [ ] #004 **DDR5 32GB (2×16GB) の具体的なモデルと価格が未確定のままです** ⬅ モデル未決定
  パーツ表に「DDR5 32GB」とあるが、メーカー・型番・AMD EXPO 対応有無・価格がすべて未定。9800X3D（AM5）は DDR5 EXPO 対応メモリで最大性能を発揮する（非 EXPO でも動くが性能が出ない場合がある）。Corsair Vengeance DDR5-6000 32GB（AMD EXPO）あたりが定番だが、価格と在庫を確認しておく必要がある。

## 💭 言語化されていない感覚・体験

- #007 **「優先確保」と「一括購入」の間で迷いが生じるかもしれません**
  次のアクションに「GPU を優先確保 → マザー+ケース+電源 → CPU+RAM+SSD」と段階が書いてあるが、分割購入は送料・管理の手間が増え、パーツが揃うまで組み立てられないもどかしさがある。GPU の品薄が本当に深刻でないなら、まとめて注文できる店（ドスパラ・パソコン工房）で一括購入する方が体験として楽かもしれない。「分割確保」か「一括注文」か、購入体験として事前に決めておくとよい。

- #008 **「購入したら即組み立てたい」気持ちと「BIOS 作業」の間にギャップがあるかもしれません**
  9800X3D は BIOS 更新が最初のステップになる。パーツが届いた日にすぐ組み立て開始できず、まず USB + BIOS ファイルを準備してから FlashBack という工程が入ることで、テンションが下がる可能性がある。BIOS USB は事前に自宅で準備しておくと組み立て当日のスムーズさが全然違う。
