## 核心

2020〜2026年の約6年分の**月次資産推移データ**（72ヶ月）を軸に、資産形成の実態・変曲点・今後の設計を分析する。
収支データ（household_all.csv）は振替・ATM引き出しなどのノイズが多く信頼性が低いため、**資産の純増減（asset_monthly.csv）を主要データとする**。

---

## データ概要

### 主要データ: asset_monthly.csv（72ヶ月分）
マネーフォワード「資産推移（月次）」エクスポート。月末残高ベース（時価）。全金融口座をMF連携済み。

| カラム | 内容 |
|---|---|
| 合計 | 総資産（円） |
| 前月比 | 前月末比増減（市場変動含む） |
| 預金現金 | 預金・現金・暗号資産 |
| 株式現物 | 株式（現物） |
| 投資信託 | 投資信託（NISA含む） |
| 年金 | iDeCo（2022/02〜開始） |
| ポイント | MF連携ポイント残高（投信購入に充当済みのため減少傾向） |

### 補助データ: household_all.csv（8,418件）
マネーフォワード入出金明細。振替・ATM引き出しが混在しており収支分析の信頼性は低い。  
**カテゴリ別支出の参考・積立パターンの特定**に利用。

## 注目点・変曲点

### 2022年4月: -574万円（最大月次下落）→ **住宅購入確定**
現預金が617万円→60万円に急減。購入価格4,620万円のマンション取得に伴う頭金・諸費用の支出。
2022年末の現預金97万円まで引き続き低水準が続いた。

### 2022年通年: -244万円（唯一の年間マイナス）
住宅購入関連支出が集中した年。

### 2023年7月: -158万円 → **旧車頭金**
現預金が300万→133万円へ急減。以前所有していた車の頭金の支出。
その後、車売却により112万円が返金。売却完了まで手元現金は180万円程度だった。
**現在の現預金287万円には、この売却益112万円が含まれている（常態は200万円前後）。**

### 2023年: +551万（最大年間増）
一時的な大型収入（事業収入・売却益等）の可能性。

### 投信の急成長（NISA活用）
2022年末: 126万 → 2026年6月: 871万（約7倍）。
株→投信への意図的な切り替え＋旧一般NISA満期による乗り換えが主因。
2024年からの新NISAで積立上限が拡大（年360万円）しており、今後さらに加速する可能性がある。

### 月次の「前月比マイナス」は支出ではなく市場変動が多い
2025/02: -105万（株が204万急落）、2025/04: -68万（株が41万減）など。
「支出が多い月」ではなく相場下落による評価損の月。
**現預金の増減のみを見ることで、実際のキャッシュフローを把握できる。**

## ⚡ 重要な未解決課題

### 現金比率問題
現預金は2020〜2026年を通じて200〜400万円台でほぼ横ばい。
これは問題ではなく、**投信積立（NISA）・プール積立・住宅ローン返済へのキャッシュ変換が設計として機能している**。
ただし手元現金の「適正水準」が未定義であり、今後の設計に必要。
子供追加・車ローン・変動金利上昇を加味した現金目標額を設定すべき。

### 車ローン繰り上げ返済の是非（要決断）
- 現在の現預金287万円のうち**112万円は旧車売却益**（常態200万円程度）
- EV購入に伴うローン: **残債620万円・金利1%・CEV補助金127万円受領待ち**
- 補助金受取後に繰り上げ返済する案があるが、**金利1%ならNISA継続（期待リターン4〜7%）の方が期待値は高い**
- 手元現金を厚くした上で慎重に判断すべき
- 実行した場合、手元現金は**約175万円**まで減少（常態180万円程度と同水準で危険）

### ボーナス依存の構造リスク
月収支がほぼイーブンのため、突発出費はボーナスで補填する構造。
子供追加による育休・固定費増加・医療費増がこの構造に重なると資金繰りが厳しくなりうる。

## 現在の考え

- 資産は2020年比で356%増と順調。**実質純資産は約2,820万円以上（不動産含む）**
- 教育費・住宅管理費として月18万円が「見えない資産形成」として機能している
- **現金が横ばいの正体が判明**：月11万の投資分を現金から取り崩し、半年ごとのボーナス115万で補填するサイクル。問題ではなく「投資に現金を変換する設計」
- 投信残高が急増しており、NISA活用と株→投信シフトが合理的に機能している
- iDeCo（2022/02〜）は所得控除の恩恵もあり、効率的な老後資産形成（ただし60歳まで拘束）
- **変動金利0.355%→1.05%への上昇は要注意**。今後の利上げ動向次第でローン返済額が増加
- 月収支がイーブンでボーナス依存の構造は、子供追加直前として最も注意すべき点
- 入出金明細ベースの「収支マイナス」という不安感は、資産推移ベースでは解消できる（2022年のみ本当のマイナス）
- 6年で16.7百万円増加は同世代比で優秀な資産形成
- **40歳3,500万目標は全シナリオで余裕を持って達成見込み。次のフェーズは目標を上方修正し、投資比率と現金比率の最適化へ**
- 子供が増えることで月次の生活費増加が見込まれ、育休シナリオ試算が最急務
- 腕時計530〜640万は心理的セーフティネットとして機能しているが、無保険かつ準流動性（即日現金化不可）の点を認識した上で位置づける
