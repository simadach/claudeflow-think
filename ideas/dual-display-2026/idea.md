# デュアルディスプレイ環境構築2026

**作成日**: 2026-06-24
**最終更新**: 2026-06-24
**ステータス**: 決定済み

---

## 核心

Mac Mini と将来購入予定の Windows WS を、各1本のケーブルで共有デュアルディスプレイに接続する。モニターはAuto Input Detectionで自動切り替え。

---

## 背景・動機

- Mac Mini M4（メイン）+ Windows WS（予定）の2台体制に移行する
- 既存モニター LG 32UL750-W が1台あり、もう1台追加してデュアル化したい
- デスクをケーブルで汚したくない。各コンピューターからの出力は1本に絞りたい
- モニター2台を物理的にリンク（デイジーチェーン等）させたい

### 接続方式

```
Mac Mini → TB4 → Anker Dock → USB-C ×2 → 両モニター
Windows  → DP  → MST Hub   → DP ×2    → 両モニター
```

- Mac Mini側：Thunderbolt 4ケーブル1本 → Anker TB4 Dockで2画面分岐
- Windows側：DisplayPortケーブル1本 → DP MSTハブで2画面分岐（DP 1.2以上のMST対応GPU必要）
- 切り替え：両モニターのAuto Input Detectionで自動切替

### 選定理由（32U720A-B）

- 既存32UL750-W（VA・3000:1）と**パネル特性が揃う**（同じVA・3000:1）
- SDR使用時の見え方が概ね近い
- 2025年6月発売の現行モデル（新品在庫確実）
- エルゴノミックスタンド標準付属
- USB-C PD 90W（既存の60Wより優れる）
- ヨドバシ.com / Amazon で購入可能

## 既知の懸念・トレードオフ

1. **HDR品質の差**：32UL750-W（DisplayHDR 600）と32U720A-B（HDR10のみ）でHDR品質が揃わない。HDRコンテンツでは差が出る。SDR使用が主体なら許容範囲と判断。（HDR は対応コンテンツを見るときだけ ON にする運用のため、常時差が気になることはない）

2. **Auto Input Detection依存**：切り替えはモニター側機能に依存。応答速度は実機確認が必要。**AID の切り替えには 5〜15 秒かかり、2台のモニターが同時に切り替わらない場合がある。** 遅延が気になる場合は AID を無効化してジョイスティックボタンで手動切り替えする運用に切り替えることも想定しておく。

3. **DP MSTハブ品質リスク**：安価品は4K/60Hz安定性にばらつきあり。StarTech・Club3D等の実績ブランドを推奨。**BIOS/UEFI やロック画面は MST Hub が OS ドライバー読み込み前は機能しないため、起動直後は片方のモニターにしか映らない場合がある。**

4. **Windows WS未購入**：現時点でWindowsは存在しない。WS購入時にGPUのDP MST対応確認が必要。RTX 5070 は通常 DP × 3 を持つため **MST Hub が不要になる可能性がある**（WS 購入後に確認してから MST Hub を購入する）。

5. **Anker Dock 周辺のケーブル本数**：「各 PC からケーブル 1 本」は達成されるが、Mac 側は Dock 経由で「Mac→Dock」「Dock→モニター1」「Dock→モニター2」「Dock 電源アダプター」の合計4本のケーブルが発生する。ケーブルトレーや配策を事前に計画しておく。

6. **両PC同時稼働時の意図しない AID 切り替え**：Mac 使用中に Windows を起動するタイミングで、Windows の映像信号発生を AID が検出して Mac のモニターが突然 Windows に切り替わる可能性がある。AID の設定を「Auto」から「Manual（手動優先）」または「Last Input（最後の入力優先）」に変更することで回避できる場合が多い。購入後すぐに AID 設定を確認・調整する。

---

## 前提条件

- Windows WS購入時のGPUがDP 1.2以上のMST対応（現行GeForce/Radeonはほぼ全対応）
- Mac Mini M4のThunderbolt 4ポートでUSB-C映像出力2画面が可能（仕様上対応）
- LG 32U720A-BのAuto Input DetectionがTB4/USB-C信号消失を正しく検出できる

---

## 意識的に考慮したが却下した観点（対象外と判断した指摘を含む）

**Thunderboltデイジーチェーン（1本で2台）**：
モニター同士をTB4でデイジーチェーンする構成も検討。LG 32UL750-WにはThunderbolt入力がないため非対応と判明。断念。

**32UL950-W（2台目）の追加**：
デイジーチェーン対応のTB3モニターを追加する案を検討。32UL750-WとのTB3→USB-CのデイジーチェーンはTB3信号をUSB-C（DP Alt Mode）が受信できないため不可と判明。断念。

**Dell U2723DE（Thunderbolt hub monitor）×2**：
TB4 + DP MST両対応の本命だが、¥80,000〜100,000/台で2台計¥200,000超。予算超過のため断念。

**LG 32UN650-W / 32UN950-W 中古**：
既存モニターとデザインを揃えるため検討。中古市場を調査したが在庫がほぼ存在せず断念。

**32U631A-B**：
安価（¥30,250）で検討したが4K非対応（WQHD）・USB-C PD 15Wのみで用途に不十分と判断。

**HDR 常時ON による2台間の表示差（#010）**：
32UL750-W（DisplayHDR 600）と 32U720A-B（HDR10）の HDR 品質差が気になるリスクを指摘されたが、HDR は対応コンテンツを見るときだけ ON にする運用のため、常時の表示差は問題にならないと判断。対象外。
