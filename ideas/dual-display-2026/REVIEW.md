## 🟡 ユーザー追加質問（2026-06-25）

- [x] #011 **Anker Prime Docking Station は見た目が好みだが、このシステムで使えるか？**
  Anker Prime は Anker の最上位ドックシリーズで、デザインが洗練されている。ただし現行モデルは **Thunderbolt 5** 世代（A83B5、約¥40,000〜60,000）であり、比較表の「Anker PowerExpand 5-in-1（TB4、¥24,980）」とは別製品。
  Mac Mini M4 の Thunderbolt ポートは TB4 なので TB5 ドックは後方互換で動作するが、TB5 の速度メリットは得られない。
  一方、Anker Prime は **HDMI 2.1 + DisplayPort 2.1** の両出力を持つため、DP MST Hub なしで 32U720A-B + 32UL750-W の **デュアルディスプレイが直接可能**（現行 Anker 5-in-1 は TB4 + HDMI のみ）。
  なお Anker Prime Dual Display モデル（A83B6、160W、¥39,800前後）も存在し、こちらは TB4/TB5 混在環境向けに最適化されている。

- [x] #012 **DP分岐器＋ドックの二重ハードウェア構成が気になる**
  「ドック ＋ DP MST Hub の2台置き」が見た目・ケーブルの煩雑さとして気になっているとのこと。
  **現在の計画では DP MST Hub は Mac Mini デュアル用には不要**：Anker 5-in-1（TB4 + HDMI）で 32U720A-B と 32UL750-W の 2台に直接つながる。DP MST Hub は将来の WS 導入後まで保留している（#009 ✅ 確認済み）。
  もし Mac Mini 使用中に「ドック 1台だけでスッキリ」を優先するなら Anker Prime（HDMI + DP 両搭載）も選択肢になる。デザインと配線の統一感を重視するなら #011 の Anker Prime を再検討する価値がある。
  💭 感覚: 「1台ドックでケーブルを最小化したい」という本質的なニーズかもしれません
  **返答より確定した要件**: 「ドック → 各モニター 1本ケーブル直結」を理想とする。分岐器・切り替え器などの中間機器を追加することは不本意。この要件は Anker 5-in-1（TB4 + HDMI）でそのまま満たせる（32U720A-B は TB4 USB-C、32UL750-W は HDMI、各1本）。
  > 返答:このせんでいきたい。分岐器、切り替えきがふえるのは不本意ドックからモニターもそれぞれ1本のケーブル接続が理想
